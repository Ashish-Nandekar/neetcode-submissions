# NeetCode 250 Journey 🚀

## About This Repository

This repository contains my solutions to the **NeetCode 250** roadmap.

I'm using this challenge to improve my Data Structures and Algorithms skills, become a better problem solver, and prepare for coding interviews. Instead of just solving problems and moving on, I wanted a place to track my progress and look back at the approaches I've learned.

## Why I'm Doing This

As a Computer Science student, I want to build strong fundamentals in problem solving and algorithmic thinking.

My goals are to:

* Complete all 250 problems
* Improve my understanding of DSA concepts
* Get better at identifying problem patterns
* Write cleaner and more efficient code
* Prepare for technical interviews
